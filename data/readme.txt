steps for organizing correctly items in the current folder (data)
- go to the official web page for the 102 flowers dataset -> https://www.robots.ox.ac.uk/~vgg/data/flowers/102/
- download the dataset images, extract images from the compressed tar archive "102flowers.tgz" into the jpg folder
- download the image labels, place the labels file "imagelabels.mat" in the current folder (data)
